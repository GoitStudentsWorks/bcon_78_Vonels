# Назва проєкту ArtistHub
Короткий опис проєкту

ArtistHub — це веб-платформа,
створена для того, щоб користувач міг швидко та зручно знайти повну інформацію про музичну групу або виконавця.

## Проєкт та його призначення

Збирає та відображає актуальну інформацію про музичні гурти та співаків.

Дозволяє знайти біографію, дискографію, фото, рейтинги та інші важливі дані в одному місці.

Допомагає людям, які хочуть отримати всю необхідну інформацію про улюблених артистів без зайвого пошуку.

Вирішує проблему розпорошених даних та робить процес пошуку максимально простим і зручним.
