# Назва проєкту ArtistHub

Короткий опис що це за проєкт.

Проект создан для того щоб користувач міг знайти всю інформацію про группу або півця.

## Про проєкт та яку задачу він вирішує


-Проект сбирає інформацію про группу або співака 
-Проект призначений для людей які бажають знайти всю необхідну інформацію про співаків в одному місці
-Проект вирішую проблеми с пошуком, а також він дуже зручний у використанні
